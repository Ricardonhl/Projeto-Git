### Aqui é o lugar para fazer anotações sobre o projeto

## Quando fazemos alterações no projeto, devemos subir no GitHUb novamente para que as alterações apareçam lá

## Se for subir os arquivos para o GitHub pela 1° vez, basta dar os comandos:

git init

git add .

git commit -m "nome do commit"

git remote add origin https://github.com/Ricardonhl/Projeto-Git.git

git push -u origin master

## Se for subir os arquivos para o GitHub pela 2° vez em diante, basta dar os comandos:

git init

git add .

git commit -m "nome do segundo commit"

git push
